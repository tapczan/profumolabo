PW Instagram feed documentation

Installation

Via FTP (recommended)

1. Unzip the pwinstafeed from the correct 1.6 or 1.7.2.4+ folder.
2. Place the folder in the root/modules/ folder.
3. Go to the "Modules" section in the back office and search for "pwinstafeed".
4. Hit "Install" and go ahead and configure the module to your liking.

Via back office (not recommended in 1.7.x)

1. Go to the "Modules" section in the back office and hit "Add new module".
2. Browse for the pwinstafeed.zip file and hit "Upload this module".
3. Go to the "Modules" section in the back office and search for "pwinstafeed".
4. Hit "Install" and go ahead and configure the module to your liking.

Instagram setup

Access token

Follow the link in the module, under the "Instagram setup" tab, "Generate an access token" grant our app access — you might be prompted to log in — and an Access Token will be generated for you.

Simply paste that into the field in the module configuration page. Make sure you copy the entire token and don't have an extra space before or after it.

Hit "Save all" and your done!

Feed settings

This entire section is for the following hook positions: Home, footer, left and right column.

Number of photos

This decides how many photos to fetch from Instagram. The Instagram API has a limit of 20 photos per request so even if you put a higher number than 20, 20 is the number that will be used.

Feed style

The style of the feed. Square, rounded (5px border radius), or circle (completely round).

Spacing

This controls the spacing between each photo in your feed.

Carousel

Whether or not to use and activate the carousel functionality.

Infite loop

If the carousel should continue to loop around when the user reaches the end, instead of just stopping.

Breakpoints

These breakpoints work from small to large. This way you can decide how many photos (or carousel items) you wish to display depending on the current breakpoint.

Extra small - From 0px upwards

Small - From 480px upwards

Medium - From 768px upwards

Large - From 992px upwards

Extra large - From 1180px upwards

Number of columns

If you're not using the carousel, this will more or less be the same as the breakpoint settings above. The number of columns in different breakpoints, working from small to large.

Extra small - From 0px upwards

Small - From 480px upwards

Medium - From 768px upwards

Large - From 992px upwards

Extra large - From 1180px upwards

Hover background color

What background color to use when you hover over the photo. You can use hex and rgba.

Hover text color

What text color to use when you hover over the photo. You can use hex and rgba.

Show likes

Whether or not to display how many likes each photo have got on hover.

Show comments

Whether or not to display how many comments each photo have got on hover.

Modal functionality

By default, the photo is linked to it's page on the Instagram site. With the modal functionality you can open it up, in a modal, and you will then see the embedded version of that photo.

Hook position

Where you want this particular feed hooked.

Custom page

Page title

Self explanatory, this will be the title of the page.

Breadcrumb title

Self explanatory, this will be the breadcrumb title of the page.
